# Professional School Web - Premium School Management System

A modern, premium frontend UI design for a Nigerian secondary school digital management system. Built with React, Tailwind CSS, and Framer Motion.

## 🎓 Features

### Pages & Sections

1. **Homepage** - Stunning hero section with animations, statistics, about preview, why choose us, testimonials
2. **About Page** - School history, mission/vision, core values, leadership team
3. **Admissions Page** - Requirements, timeline, tuition rates, application form, FAQs
4. **News & Events Page** - School announcements, events calendar with filtering
5. **Payment Portal** - Fee overview, payment methods, history, payment tracking
6. **Parent Portal** - Secure login with demo credentials
7. **Parent Dashboard** - Student profile, academic summary, notifications, fee status
8. **Results Portal** - Secure result access with code verification, detailed grades
9. **Teacher Portal** - Upload results, manage assignments, student records
10. **404 Page** - Beautiful error page

### Design Features

- ✨ **Beautiful Stock Images** - Real Nigerian and African educational imagery from Unsplash/Pexels
- 🎨 **Premium Design** - Deep blue primary color, gold/orange accents, glassmorphism
- ✅ **Smooth Animations** - Framer Motion animations throughout
- 📱 **Fully Responsive** - Mobile, tablet, and desktop optimized
- 🌙 **Dark Mode Toggle** - Switch between light and dark themes
- 📊 **Real Charts** - Recharts for academic performance visualization
- 🔐 **Result Access Code System** - Secure result verification UI
- 💳 **Payment Integration UI** - Multiple payment methods (Paystack, Flutterwave, Bank Transfer)
- 🎯 **Interactive Components** - Loading states, success modals, notifications

## 🛠️ Tech Stack

- **React 18** - UI framework
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **Recharts** - Charts and graphs
- **React Router** - Navigation
- **Vite** - Build tool

## 🚀 Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn

### Installation

1. Clone or navigate to the project directory:

```bash
cd "school template"
```

2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

The application will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```
